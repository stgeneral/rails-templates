Rails Templates
===============

Frontend Project
---------
* create a project
<pre>
rails new myapp --skip-active-record --skip-test-unit -m https://raw.github.com/stgeneral/rails-templates/master/frontend.rb
</pre>
* sart development server
<pre>
rails s
</pre>
* build web app
<pre>
rake front:build
</pre>
